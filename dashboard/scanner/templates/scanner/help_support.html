Top Questions

To scan your network for vulnerabilities:

  1. Navigate to the Overview page
  2. Click on the "Scan Network" button in the top navigation bar
  3. Wait for the scan to complete - this may take several minutes depending on your network size
  4. Review the results in the Vulnerabilities section

For more detailed scans, you can configure advanced options in the Settings page.

Rex Security uses a 4-level risk classification system:

  • Critical - Severe vulnerabilities that require immediate attention. These can lead to system compromise with minimal effort.
  • High - Serious vulnerabilities that should be addressed promptly. These pose significant risk to your systems.
  • Medium - Moderate vulnerabilities that should be fixed in your regular maintenance cycle.
  • Low - Minor vulnerabilities with limited potential impact.

The recommended scanning frequency depends on your security requirements:

  • Daily scans - For high-security environments or systems with frequent changes
  • Weekly scans - For most business environments (recommended default)
  • Monthly scans - Minimum recommendation for stable environments

Premium users can set up automated scanning schedules in the Settings page.

For each vulnerability detected, Rex provides:

  1. A detailed description of the vulnerability
  2. The potential impact if exploited
  3. Step-by-step remediation instructions
  4. Links to relevant security advisories

Premium users get access to automated remediation for many common vulnerabilities.
